#### 参数
NOTICE_TYPE
- feishu: 使用飞书发送通知
- dingtalk: 使用钉钉发送通知
- workWechat: 使用企业微信发送通知
- showDoc: 使用ShowDoc发送通知

MSG_TYPE
- text: 发送纯文本通知
- markdown: 发送 Markdown 格式的通知
- card: 发送卡片式通知
